Selected R Code files from different papers/projects

    López-Méndez, M., Ospina-Escobar, A., Iskandar, R., & Alarid-Escudero, F. (2021). Age-specific rates of onset of cannabis use in Mexico. Addictive         behaviors, 122, 107038.PMCID: PMC8645182. DOI: 10.1016/j.addbeh.2021.107038
   
    For raw data files please contact me at mlopezme@gmail.com. 
    The article is freely available at https://www.ncbi.nlm.nih.gov/pmc/articles/PMC8645182/
